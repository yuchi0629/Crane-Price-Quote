# 中联塔机报价单生成软件

这是一个桌面版报价单生成工具。业务员可以通过现有 Excel 导入内置数据库，选择语言、机型、安装形式、贸易术语、价格和增减配项，然后生成中联重科格式的塔机报价 PDF。

## 运行

```powershell
cd "C:\Users\13973\Documents\中联塔机报价单程序"
python app.py
```

如果本机缺少依赖：

```powershell
python -m pip install openpyxl reportlab
```

## 数据导入

研发配置导入功能位于界面左下角的 `研发配置导入模块`，需要输入工号、时间和密码 `zlzk.123456789` 后才能进入。进入后包含 3 个导入按钮：

1. `导入增减配价格表`
   - 导入 `PriceInDecreasePrice_dateOut.xlsx`。
   - 只录入 D 列“产品组”为 `B1` 的行。
   - 识别 E 列产品型号、G 列产品编码，并保存一对一关系。
   - 识别 F 列配置描述里的类型、臂长、吨数、塔身种类。
   - 同时录入该产品编码下的增减配项。

2. `导入产品安装参数表`
   - 导入 `PrdTowerCraneCfg_dataOut.xlsx`。
   - A 列是产品编码。
   - 读取安装形式、塔高、臂长、容绳量等信息。

3. `导入当前机型配置表`
   - 点击后先选择这张配置表对应的产品型号，再选择 Excel 配置表。
   - 软件按你选择的产品型号入库，不再根据文件名或表头自动猜型号。
   - 如果再次导入同一个机型的配置表，会覆盖该机型之前的配置表数据，不会同时保留多份。
   - 自动识别安装形式列。
   - 只读取该安装形式下带实心圈 `●` 的主要部件。

左下角还有 `查看机型配置表导入状态`：
- 显示每个产品型号是否已经导入同名配置表。
- 配置表按产品型号严格匹配，例如 `R220-10` 不会用于 `R220-10RA(CE)`。
- 显示导入文件名、匹配到的安装形式、可见主要部件数量。

左下角还有 `配置导入修改记录`：
- 显示每次研发配置导入的时间、工号、更改内容。
- 记录内容包括导入文件、导入数量，以及机型配置表是新增还是覆盖旧配置。

## GUI 使用

界面分为以下区域：

- 报价单信息：报价单语言、报价日期、贸易术语、交易地点、价格、价格单位、报价单位、报价人、联系电话、联系邮箱、公司地址。
- 产品信息：产品型号、安装形式。
- 产品型号下拉和配置表状态窗口不显示 `D` 开头或 `ZT` 开头的型号。
- 交易条款及其他信息：付款方式、交货期、报价有效期、运输方案、质保期、其他。
- 产品基本信息：根据数据库自动识别产品编码、塔机类型、独立高度、最大臂长、最大起重量、容绳量、塔身种类、主要部件数量。
- 可选增减配置：选择增减配项和每项数量。

顶部按钮：
- `生成报价 PDF`：按当前机型、安装形式和报价信息生成报价单。
- `生成基本配置清单`：按当前机型和安装形式生成 Excel，只保留配置表中该安装形式列带实心圈 `●` 的行；删除原配置表顶部两行标题，不写来源配置表；删除原 B 列“组成”；第 4、5 行按列合并并保留有数据的单元格；第一列从第一个部件开始重新按 `1、2、3、4` 编号；塔身等项目保留原配置表中的 `●`、`●×数量`；删除编码列、备注列、重量列。
- `生成增减配清单`：按当前机型和安装形式生成 Excel，保留配置表中该安装形式列带空心圈 `○` 的行，以及名称含标准节、基节、基础节、底架、支腿座、基础预埋、支腿组件的行，并删除序号列、编码列、备注列、重量列。
- 关键词行改空心圈规则只作用于增减配清单，不影响基本配置清单。
- 两类清单会尽量保留原配置表中仍然完整存在的合并单元格。增减配清单中因关键词保留的行，当前安装形式列标记会统一改为 `○`。
- 新版导入机型配置表时，会把原 Excel 内容内嵌保存到数据库里；后续生成清单直接使用已导入配置表，不需要重复导入。旧版本摘要数据缺少原 Excel 内容时，软件会尝试按已记录文件名自动查找原配置表并补建清单数据。

点击 `生成报价 PDF` 后，文件会输出到：

```text
output/pdf/
```

报价单位、报价人、联系电话、联系邮箱、公司地址会自动记忆。关闭软件或生成报价单时会保存到 `data/user_settings.json`，下次打开自动沿用上次填写的内容。

## 自动识别规则

- `LW` 开头：风电塔机
- `RL` 开头：平头动臂塔机
- `LR` 开头：桅杆吊塔机
- `L` 开头：动臂塔机
- `T`、`W`、`WA`、`R` 开头：平臂塔机
- 最大起重量：读取产品型号中 `-` 后面的数字，例如 `R220-10RA(CE)` 识别为 `10t`

## 报价单内容

生成的 PDF 包含：

1. 开头：左侧 `zoomlion.png`，右侧可编辑地址，默认 `Zoomlion Smart City Headquarters Building, No.613 NaqiuRoad, WangCheng District, Changsha, Hunan, China`。
2. 报价信息：报价单位、报价人、联系电话、联系邮箱。默认报价单位为“中联重科建筑起重机械公司”。
3. 基本参数：产品型号、塔机类型、独立高度 HUH、最大臂长、最大起重量、安装形式、单价。
4. 主要部件配置：起重臂、平衡臂、司机室、上支座、回转支承、下支座、吊钩、标准节、底架、支腿组件、基础预埋、机构、钢丝绳、电控总成；名称中含“插头电缆包”或“图”的部件不展示。
5. 增减配置：由业务员勾选，报价单中不显示增减配价格。
6. 交易条款及其他信息：由业务员在 GUI 中填写。

## 多语言词典

软件内置了当前参考表中已出现中文词汇的英文、法语、德语翻译，包括固定栏目标题、交易条款标题、塔机类型、安装形式、塔身种类、主要部件名称、增减配类别和部件名称。

选择 English、Français 或 Deutsch 后，报价单会统一输出为对应语言。后续如果导入的新 Excel 出现词典里没有的新中文词，软件会暂时保留原文，可以继续在 `app.py` 的 `TRANSLATIONS` 里补充。
